## Quick Start

### Prerequisites

| Tool | Version | Install |
|------|---------|---------|
| Python | ≥ 3.8 | https://python.org |
| Node.js | ≥ 18 | https://nodejs.org (LTS) |
| NumPy | optional | `pip install numpy` (3× faster QFT) |

### Installation

```bash
# 1. Clone
git clone https://github.com/sansqrit/sansqritPy.git
cd sansqritPy

# 2. Install Node.js dependencies
npm install

# 3. Run pre-computation (generates all 8 data files)
python3 precompute.py

# 4. Verify all files generated
python3 precompute.py --check

# 5. Run all tests
npm run test:all   # must show 71/71 passing

# 6. Start
npm start
# → 📦 528 blocks loaded across 42 categories
# → Server running at http://localhost:3000
```

## The Python Pre-computation Engine

File: `precompute.py` (1,248 lines, pure Python, no dependencies beyond stdlib)

### What It Computes

#### 1. Gate Matrices (`gate_matrices.json` — 8 KB)

Pre-computes all 2×2 unitary gate matrices with exact complex values:

```python
# Every gate is verified: U·U† = I (unitarity check)
gates = {
    "I": [[1,0],[0,1]],
    "X": [[0,1],[1,0]],
    "Y": [[0,-i],[i,0]],
    "Z": [[1,0],[0,-1]],
    "H": [[1/√2, 1/√2],[1/√2, -1/√2]],
    "S": [[1,0],[0,i]],
    "T": [[1,0],[0,e^(iπ/4)]],
    # + Rx(θ), Ry(θ), Rz(θ) at 11 common angles
    # + 9 composed gate pairs: H·X, S·S, T·T, etc.
}
```

#### 2. QFT Matrices (`qft_matrices.json` — 4 MB)

Exact QFT unitary matrices for n=1..8 qubits:

```
M[j][k] = (1/√N) · e^(2πi·j·k/N)   where N = 2^n
```

Each matrix is verified: first row is all 1/√N (real). Used by the engine to
compute exact QFT without running the circuit gate by gate.

#### 3. Molecule Hamiltonians (`molecule_hamiltonians.json` — 4 KB)

Pauli-string Hamiltonians from Jordan-Wigner transformation (STO-3G basis):

```
H = Σᵢ cᵢ · Pᵢ

H₂: -0.810548 IIII + 0.172184 IIIZ + ... (15 terms)
LiH: -7.496538 IIII + ... (14 terms, 4-qubit active space)
BeH₂, H₂O: similar structure
```

Plus H₂ dissociation curve: 20 bond lengths from 0.40 Å to 5.00 Å.

#### 4. Pauli Table (`pauli_table.json` — 7 KB)

Complete Pauli group multiplication: `{I, X, Y, Z}` forms a group.

```
X·Y = iZ, Y·Z = iX, Z·X = iY, X·X = I, etc.
```

Includes: commutation relations, eigenvalue tables, Z-basis sign table for VQE.

#### 5. Grover Table (`grover_table.json` — 18 KB)

Optimal Grover iteration counts for all (nQ, M) pairs:

```
iters = round(π/4 · √(N/M))
prob  = sin²((2·iters+1) · arcsin(√(M/N)))

nQ=4, M=1: iters=3, success_prob=0.9613
nQ=8, M=1: iters=12, success_prob=0.9999
```

#### 6. Noise Kraus Operators (`noise_kraus.json` — 40 KB)

5 standard noise channels at 13 error rates (0.001 to 0.10):

| Channel | Formula | Kraus Operators |
|---------|---------|----------------|
| Depolarising | ρ′ = (1-p)ρ + (p/3)(XρX+YρY+ZρZ) | K₀=√(1-p)I, K₁=√(p/3)X, K₂=√(p/3)Y, K₃=√(p/3)Z |
| Amplitude Damping | T1 decay \|1⟩→\|0⟩ | K₀=[[1,0],[0,√(1-γ)]], K₁=[[0,√γ],[0,0]] |
| Phase Damping | T2 dephasing | K₀=[[1,0],[0,√(1-λ)]], K₁=[[0,0],[0,√λ]] |
| Bit Flip | X error | K₀=√(1-p)I, K₁=√p·X |
| Phase Flip | Z error | K₀=√(1-p)I, K₁=√p·Z |

Plus 189 pre-computed T1/T2 decay values for common gate times.

#### 7. Circuit Identities (`circuit_identities.json` — 4 KB)

Gate simplification rules for circuit optimisation:
- Self-inverse: H·H=I, X·X=I, CNOT·CNOT=I
- Power rules: T·T=S, S·S=Z
- Basis changes: H·Z·H=X, H·X·H=Z, S·X·S†=Y
- Rotation merging: Rx(a)·Rx(b)=Rx(a+b)
- Toffoli decomposition: 6 CNOTs + 7 T gates (Selinger 2013)

#### 8. Molecular Energies (`molecular_energies.json` — 2 KB)

FCI/HF/CCSD reference energies for H₂, LiH, H₂O, NH₃, N₂, BeH₂:

```
H₂ @ STO-3G: HF=-1.117099, FCI=-1.137270, exp=-1.174476 Hartree
H₂O @ STO-3G: HF=-74.963002, FCI=-74.965901 Hartree
```

### CLI Usage

```bash
# Generate all 8 files (~0.7 seconds, stdlib only)
python3 precompute.py

# Faster with NumPy (pip install numpy)

# Verify existing files

# Delete and regenerate from scratch
python3 precompute.py --clean

# Generate only specific tables
python3 precompute.py --target gate_matrices
python3 precompute.py --target qft

python3 precompute.py --target pauli
python3 precompute.py --target grover
python3 precompute.py --target noise
python3 precompute.py --target identities
python3 precompute.py --target molecular_energies
```

### Python API

```python
import json

# Load Grover lookup table
with open("precomputed/grover_table.json") as f:
    grover = json.load(f)

# Find optimal iterations: 4-qubit register, 1 target
entry = grover["table"]["4"]["single_target"]
print(f"Optimal iterations: {entry['optimal_iters']}")    # 3
print(f"Success probability: {entry['success_prob']}")    # 0.9613

# Load molecule Hamiltonian
with open("precomputed/molecule_hamiltonians.json") as f:
    mols = json.load(f)

h2 = mols["hamiltonians"]["H2"]
print(f"H2 FCI energy: {h2['fci_energy_hartree']} Ha")   # -1.137270
print(f"Pauli terms: {len(h2['terms'])}")                  # 15
for term in h2["terms"][:3]:
    print(f"  {term['coefficient']:+.6f} × {term['pauli']}")

# Load gate matrices and verify unitarity
with open("precomputed/gate_matrices.json") as f:
    gates = json.load(f)

h_gate = gates["gates"]["H"]  # [[a,b],[c,d]] complex entries
print(f"H[0][0] = {h_gate[0][0]}")   # {re: 0.7071, im: 0.0}

# Load noise Kraus operators
with open("precomputed/noise_kraus.json") as f:
    noise = json.load(f)

# Get depolarising channel at p=0.01 error rate
k = noise["channels"]["depolarising"]["kraus_by_rate"]["0.01"]
print(f"K0 = {k['K0']}")
print(f"K1 = {k['K1']}")
```

## Sanskrit DSL — Quick Reference

```sanskrit
-- Variables
let x = 42
mut counter = 0

-- Quantum operations
let q = qubits(4)
H(q[0]); CNOT(q[0], q[1])
let r = measure_all(q, shots=1000)
print(r.histogram)

-- VQE Chemistry
molecule H2 { atoms:[H,H], bond_length:0.74, basis_set:"STO-3G" }
let result = vqe(H2, ansatz="UCCSD", shots=2000)
print(f"Energy: {result.energy:.6f} Ha")

-- Statistics (200+ functions)
let data = [2.3, 4.1, 3.8, 5.2, 4.7]
print(f"Mean: {mean(data):.4f}")
print(f"Std:  {stdev(data):.4f}")
print(percentile(data, 95))
```
