# Sanskrit Visual Builder v3.2
## ETL + Data Science + Quantum

### Run
```
node src/server/server.js
```
Open http://localhost:3000

### No npm install! Pure Node.js built-ins.

### What's included
- 528 Quantum blocks (42 categories)
- 28 ETL operations: CSV/JSON reader, filter, join (all types), group by, pivot, unpivot, normalize, encode, deduplicate, validate, profile, and more
- 7 Data Science ops: normalize, encode, train/test split, linear regression, correlation, describe, outlier detection
- Rich visualizations: histogram, pie, bar, line, scatter, heatmap, statevector, gauge, sortable/filterable table
- Settings panel with theme, font size, auto-save, shortcuts
- 50+ visual block examples (ETL + DS + Quantum)
- File upload drag & drop
- Video screen recording
- .sanskrit save/load format
- Live CPU/memory metrics via SSE
